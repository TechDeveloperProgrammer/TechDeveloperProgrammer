# 💫 About Me:
🌟 Christian LEGIONARY 🌟<br><br> Saludos, mi nombre es **Cristian Adrián Ruiz González**, un estudiante apasionado de **Ingeniería en Computación** en **UNIJAL**, Jalisco.
